咦? 后面好像有些东西乱入了....对,接下来讲这些乱入的东西,弯曲的箭头是如何实现的?Json说: 用纯CSS实现嘛? 这样子在标签里面再加一个标签,利用里标签的伪类和圆角效果就可以做出这样子的东西出来. 上代码:

.curve{
	height: 10px;
	width: 15px;
	top: 50%;
	left: -10%;
	border-bottom-left-radius:50px;
	border-left:12px solid #aaa;
	border-bottom:12px solid #aaa;
}
span{
	width:40px;
	height:40px;
	float:left;
	position: relative;
	margin-left:20px;
}

平姨摇了摇头,这个明明就可以用一个标签来实现,(卖萌地)说了一声:"看我的!":

.challenge i{
	display: inline-block;
	margin: 40px;
	width: 0;
	height: 0;
	font-size: 0;
	line-height: 0;
	border: 20px solid #AAAAAA;
}
.challenge .arrow-s7{
	position: absolute;
	width: 12px;
	height: 12px;
	position: relative;
	border-left:20px solid #AAAAAA;

}
.challenge .arrow-s7:after{
	top: -18px;
